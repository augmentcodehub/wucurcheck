# wucurcheck 功能介绍

## 系统概述

wucurcheck 是一个自动化账号管理系统，通过 Cloudflare Worker 提供 Web 管理界面，配合 GitHub Actions 执行注册和签到任务。

## 架构

```
┌─────────────────────────────────────────────────────────┐
│  用户浏览器                                               │
│  https://worker-dashboard.ouraihub.workers.dev          │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│  Cloudflare Worker（管理后台）                             │
│  ├── 登录/登出                                           │
│  ├── 账号列表（KV 存储）                                  │
│  ├── 触发 GitHub Actions                                 │
│  ├── 接收回调写入 KV                                      │
│  └── 定时 Cron 触发签到                                   │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│  GitHub Actions                                          │
│  ├── register.yml      批量注册                           │
│  ├── checkin_batch.yml  批量签到（未签到账号）              │
│  ├── checkin.yml        定时全量签到                       │
│  └── 完成后 POST /callback 回写结果                       │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│  Python 后端                                             │
│  ├── 账号生成（自然用户名：水果/植物 + 数字 + 动物/金属）   │
│  ├── 注册（HTTP API 调用目标平台）                        │
│  └── 签到（登录 + 签到 + 查余额）                         │
└─────────────────────────────────────────────────────────┘
```

## 功能清单

### 1. 登录管理
- Cookie Session 认证（7 天有效）
- 本地开发兼容（SameSite=Lax）
- 未登录自动跳转登录页

### 2. 账号列表
- 展示所有账号：用户名、平台、状态、余额、签到时间、最近结果
- 密码默认隐藏（API 不返回）
- 统计摘要：总账号数、今日已签到、未签到
- 空态/错误态友好提示

### 3. 批量注册
- 页面弹窗配置：数量（1-50）、用户名组合、邮箱域名、密码
- 用户名组合：水果+动物、植物+动物、水果+金属、植物+金属
- 每个分类 50 个英文单词，中间 1 位随机数字
- 实时预览用户名样例和长度校验
- 域名过长自动禁用提交（平台限制 20 字符）
- 注册完自动签到一次
- 间隔 10 秒防止 429 频率限制

### 4. 一键签到未签到
- 自动筛选今日未签到的账号
- 传给 GitHub Actions 逐个签到
- 每个账号间隔 5-10 秒随机延迟（防风控）
- 签到完批量回调写入结果

### 5. 手动触发签到
- 单个账号签到
- 全量签到（触发 checkin.yml）

### 6. 定时自动签到
- 页面可配置签到时间（多选）
- Worker Cron 每小时检查，匹配时间点自动触发
- 默认北京时间 08:00

### 7. 批量删除
- 勾选多个账号删除
- 一键删除所有失败账号
- 一键删除全部账号
- 二次确认防误删

### 8. 导出 CSV
- 一键导出所有账号为 CSV 文件
- 包含：用户名、密码、平台、状态、余额、签到时间、最近结果

### 9. 账号详情
- 点击账号查看详情弹窗
- 展示注册时间、最后签到、余额、最近结果等

### 10. 触发锁保护
- 同一操作未完成时不重复触发（返回 IN_PROGRESS）
- 锁 5 分钟自动过期（防止异常卡死）
- 触发失败立即释放锁
- 回调成功后释放锁

### 11. 结构化日志
- JSON 格式，每个请求带 request ID
- `wrangler tail` 实时查看
- 记录所有关键操作：登录、触发、回调、删除

### 12. 多用户管理
- 基于角色的权限控制（admin / viewer）
- admin 可添加/删除用户
- 每个用户可修改自己的密码
- 用户数据存储在 KV（`user:{username}`）

### 13. 密码安全
- 页面密码默认脱敏显示（••••••），点击才显示
- 所有 secret 比较使用 timingSafeEqual（防时序攻击）
- 支持从页面修改密码（存入 KV，优先级高于环境变量）

### 14. 本地代理池（IP 轮换）
- 从 VPN 订阅链接解析节点
- 通过 sing-box 为每个节点开本地 SOCKS5 端口
- Python 注册时轮换不同代理，绕过平台 IP 限流
- 支持 sing-box JSON 和 vless/hysteria2 URI 格式
- 完全在 WSL 本地运行，不影响 Windows
